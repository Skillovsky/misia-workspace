# Misia Workspace 🐻

Workspace asystentki **Misia** - Telegram bot Jacke'a do promocji i komunikacji.

## Stack
- **Model:** Gemini Flash / Groq Llama / DeepSeek (darmowe!)
- **Channel:** Telegram
- **Platform:** Clawdbot

## Funkcje
- 🌤️ Pogoda
- 📺 Transkrypcje YouTube  
- 📝 Podsumowania
- 📅 Kalendarz
- 🔧 Google Workspace CLI

## Hierarchia
Misia → proste zadania, info, promocja
Majki (WhatsApp) → trudne pytania, analizy biznesowe
